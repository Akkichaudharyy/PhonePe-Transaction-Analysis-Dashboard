# PhonePe Transaction Analysis Dashboard

## Project Overview

This project analyzes PhonePe transaction and user data using Power BI to understand transaction performance, customer behavior, and digital payment trends.

## Objectives

* Analyze transaction volume and transaction value.
* Understand customer demographics and user distribution.
* Identify transaction patterns and user engagement trends.
* Build an interactive dashboard for business reporting.

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling

## Dashboard Features

* Transaction Performance KPIs
* User Demographics Analysis
* Age Group Distribution
* Interactive Filters and Slicers
* Trend Analysis Visualizations
* Business Insights Dashboard

## Key Skills Demonstrated

* Data Cleaning
* Data Transformation
* Dashboard Development
* Data Visualization
* Business Intelligence
* KPI Creation
* DAX Measures

## Outcome

The dashboard provides a comprehensive view of PhonePe transaction activity and customer behavior, helping stakeholders understand digital payment adoption and user engagement.
